# Measuring-50Hz
This code measures the mains frequency of 50Hz.<br>
The signal source is a SINE wave.<br>
The sine wave from the output of ZMPT101B<br>
<br>
<B>Main parts:</B><br>
ESP32 - D1<br>
ZMPT101B<br>
TFT display 240x240 - ST7789<br>
